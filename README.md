# gps_drivers

GPS Driver Management System - Flutter Web.

## Configuration

Set these environment variables before building or deploying:

- `ADMIN_PIN` - manager login PIN.
- `MAPBOX_TOKEN` - optional Mapbox token, if map styles need it.
- `GOOGLE_PLACES_KEY` - Google Places API key for the Netlify geocode function.
